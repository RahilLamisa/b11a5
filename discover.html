Question - 1 : What are the different ways to select an element in the DOM?

There are many ways to select an element in the DOM such as -
i) document.getElementById()
ii) document.getElementsByClassName()
iii) document.getElementsByTagName()
iv) document.querySelector()
v) document.querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Here is the difference between them -
1. innerHTML : It returns the html code of an element and also is used to change or set the code.
2. innerText : It returns the text inside of an element and also is used to change or set the inner text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique or pattern where parent element handles events for it's child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a pattern where an event starts from target element and then propagates to its parent elements in the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

1. Create an element : document.createElement('elementName')
2. Add an element : element.append('elementName')
3. Remove an element : element.remove('elementName')
